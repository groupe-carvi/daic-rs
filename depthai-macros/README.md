# depthai-macros

Procedural macros used by the `depthai` crate.

This crate is part of the **depthai-rs** workspace (Rust bindings + safe-ish wrapper for Luxonis DepthAI-Core v3).

- Main crate: `depthai` (high-level Rust API)
- FFI crate: `depthai-sys` (builds/links DepthAI-Core and exposes bindings)
